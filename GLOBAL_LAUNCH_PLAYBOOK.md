## Product Hunt — lansman günü

### Başlık önerisi

```
Fingofy — Budget, expenses & savings goals for Android
```

### Tagline (60 karakter)

```
Personal finance app: budget, track spending, hit your goals
```

### Açıklama şablonu

```
Fingofy helps you manage money on Android:

• Monthly budgets with category limits
• Income & expense tracking
• Savings goals with progress
• Payment / installment plans
• Charts and monthly summaries
• Dark mode, multiple languages

Core features are free. Built with React Native + Firebase.

I'd love feedback from the PH community — especially on onboarding and first budget flow.

Google Play: https://play.google.com/store/apps/details?id=com.umaycorporation.fingofy
```

## Indie Hackers — Show IH

### Başlık

```
Show IH: Fingofy — I built a free budget app for Android after struggling with spreadsheets
```

### Gönderi şablonu

```
**Problem**
I wanted a simple way to track monthly spending and savings goals without juggling spreadsheets.

**Solution**
Fingofy — budget categories, transaction log, goal progress, and payment plans in one Android app.

**Stack**
React Native, Firebase (auth, Firestore, analytics), Google Play.

**Status**
Live on Google Play. Core features free; optional IAP removes ads.

**Ask**
1. What would make you try a new finance app?
2. Is the first-budget flow clear enough?

Play Store: https://play.google.com/store/apps/details?id=com.umaycorporation.fingofy
```
