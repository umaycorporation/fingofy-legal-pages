## Hafta 1

### Pazartesi — Reels / TikTok (30 sn)

**Konu:** Aylık bütçe 3 adımda

**Senaryo:**
1. (0–5 sn) “Ay sonu parası nereye gidiyor?” — metin overlay
2. (5–20 sn) Ekran kaydı: Bütçe oluştur → kategori limitleri
3. (20–30 sn) Dashboard özeti + “Link bio’da / açıklamada”

**Açıklama metni:**
```
Aylık bütçeni 3 adımda kurabilirsin: kategori seç, limit belirle, harcamalarını takip et.

Fingofy ücretsiz — Android:
https://play.google.com/store/apps/details?id=com.umaycorporation.fingofy

#bütçe #harcamatakip #kişiselfinans
```

### Perşembe — Statik kart (Instagram carousel veya tek görsel)

**Başlık:** Harcamalarını kategorize et

**Gövde:**
```
Market, ulaşım, faturalar… Hepsini ayrı görünce kontrol kolaylaşır.

Fingofy ile giderlerini kategorilere ayır, aylık özeti tek ekranda gör.

Ücretsiz indir (Android):
https://play.google.com/store/apps/details?id=com.umaycorporation.fingofy
```
